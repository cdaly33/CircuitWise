A resistor is an electrical component that limits the flow of current in a circuit. Resistance is measured in ohms (Ω). Resistors are used to protect components, divide voltage, set timing, and generate heat. Fixed resistors have a set value indicated by color-coded bands, while variable resistors (potentiometers) can be adjusted. They are the most common component in electronics.
