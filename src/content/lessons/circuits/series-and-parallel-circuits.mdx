## Circuit Configurations

Components in a circuit can be connected in two basic ways: **series** and **parallel**.

### Series Circuits

In a series circuit, components are connected end-to-end, forming a single path for current flow.

### Parallel Circuits

In a parallel circuit, components are connected across the same two points, creating multiple paths for current.

### Key Differences

- **Series:** Same current flows through all components; voltages add up
- **Parallel:** Same voltage across all components; currents add up
